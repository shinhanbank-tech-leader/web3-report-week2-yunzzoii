# 제목: 비트코인 결제와 기존 금융 지급결제의 구조적 차이 비교

## 레포트 내용
전통적인 금융 시스템은 중앙기관을 중심으로, 은행과 결제기관을 통해 거래가 이루어져 왔다. 반면 비트코인의 경우 블록체인 기술을 기반으로 중앙기관 없이도 참여자간 직접 가치 이전을 가능하게 하고 있다. 

### 기존 금융 시스템
은행, 카드사, 중앙은행 등 신뢰할 수 있는 제3자가 모든 거래를 중개한다. 모든 장부는 은행의 중앙 서버에 보관되며, 은행이 거래의 유효성을 승인한다. 
승인 -> 정산 의 구조를 가지고 있기 때문에, 사용자가 보기에는 즉시 결제된 것처럼 보이지만, 은행 간 자금 정산은 며칠이 소요될 수 있다. 
또한, 기존 금융 시스템의 경우 개인 신원 확인을 통해 실명과 연결된 계좌를 개설해야 한다. 기관이 거래를 중개하기 때문에, 잘못된 거래를 법적으로 보호할 수 있는 소비자 보호의 기능이 있다. 

### 비트코인
중앙 관리자 없이, 전 세계에 분산된 노드들이 네트워크를 유지한다. 모든 참여자가 동일한 장부를 공유하고, 네트워크 참여자들이 공동으로 검증하여 거래의 유효성을 승인한다. 
결제과 정산이 동시에 발생하며, 비트코인이 전송되어 블록에 기록되는 순간 가치는 상대방에게 이동하게 된다. 그러나, 네트워크 혼잡도에 따라 블록 생성시간만큼의 대기 시간이 발생할 수 있다. 실명정보 대신 공개키 기반의 지갑 주소를 사용하여, 거래 내역은 모두에게 공개되지만 해당 주소의 주인이 누구인지는 암호화되어 있어 익명성을 유지한다. 결제 취소가 어려운 구조이기 때문에 스스로 보안을 책임져야 한다.
